Таблица для товаров: products
Cтолбцы: id(int), existence(text), photos(text), description(text), price(int),
feature(text)

existence указывает стадию готовности товара. То есть есть в наличии или
нужно ждать его изготовления. Могут быть и другие варианты. Также содержит
количество товаров.

photos содержит список фотографий. Некоторые из них могутиметь расширение
.hidden означает, что фотография видна только для администратора.



Таблица для категорий: categories
Cтолбцы: id(int), parent(int), type(text), list(text)

Тип category означает, что при нажатии на нее откроется список товаров.

Если тип link, значит в list не товары а другие категории как эта же link
или category. При нажатии открывается список категорий.

parent нужен для возврвта в предыдущую категорию.


Таблица для статей: articles
Столбцы: id(int), title(text), body(text)
