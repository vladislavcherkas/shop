Таблица для товаров: products
Cтолбцы: id(int), existence(text), photos(text), description(text), price(int),
feature(text)

existence указывает стадию готовности товара. То есть есть в наличии или
нужно ждать его изготовления. Могут быть и другие варианты. Также содержит
количество товаров.

photos содержит список фотографий. Некоторые из них могутиметь расширение
.hidden означает, что фотография видна только для администратора.



Таблица для категорий: categories
Cтолбцы: path(text), content(text)

Какие существуют категории?
Существуют те которые содержат другие категории и те, что содержат товар.
Если категория содержит товар, то открывается экран с товарами. Для простоты
они будут упоминаться как директории и файлы соответственно.

Что такое директория?
Это строка оканчивающияся на /. Например Товары/, Статьи/. Директория / является
корневой. Корневая директория открывается по умолчанию.

Что такое файл?
Это строка. Например "Размеры 40х90". При открытии такой директории отображается
экран с товарами.

Директория может содержать файлы и другие директории.

Как хранить папки, файлы и товары?
Папки и файлы:
/Все товары
/Синтепон/Сумки для покупок/
/Синтепон/Как стирать
/Статьи/Описание/
