RequestPOST
Класс RequestPOST предназначен для обмена POST запросами между клиентом и сервером в обоих направлениях. Принимает масив с переменными, а отдаёт строку.
Пример использования
js
function receiver(data) {
    console.log(data);
}
const data = {
    "id": "40",
    "name": "txt",
};
new RequestPOST("development.php", data, receiver);
php
echo $_POST['id'] . " " . $_POST['name'];
Рекомендуется использовать такие названия: 
function receiver(data) {}
